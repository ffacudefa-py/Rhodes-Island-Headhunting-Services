---------- RHODES ISLAND HEADHUNTING SERVICES ----------

A script that uses CustomTKinter to show an interface where you can search for any Operator released in the English server of Arknights.
It searches for the Operator's Rarity, Class, Branch, and more! (data is obtainted from Arknights's wiki.gg)
It also lets you see the selected Operator's Elite 1 and Elite 2 art.
